# Security Policy

K-WASM은 K-PaaS에 적용되는 WASM 서비스입니다.

## Supported Versions

K-WASM 앱의 지원되는 버전은 다음과 같습니다.

| Version | Supported                                    |
| ------- | -------------------------------------------- |
|  0.01   | Frameworks : Spring Boot Frameworks 3.16
|         |  Language : Java 17
|         |   OS : CentOS 8, Rocky Linux 8, Ubuntu 18.04
